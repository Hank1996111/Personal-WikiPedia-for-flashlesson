Perplexity AI 於 2026/6/17 推出 Brain，專為 AI Agent 平台「Computer」設計的自我改善記憶系統。

## 核心機制
- Context Graph（脈絡圖譜）：記錄 Agent 的工作成果、資料來源、用戶修正行為
- 隔夜自我更新：每天整合 session 記錄，刷新 LLM wiki
- 下次任務前預載入，讓 Computer 帶著完整記憶開始工作

## 效能提升
- 答案正確率 +25%
- 記憶召回率 +16%
- 需要歷史脈絡的任務成本 -13%

## 使用方式
- 研究預覽版，限 Perplexity Max ($200/月) 與 Enterprise Max 訂閱者
- 側邊欄 Customize 開啟
- 目前無法完全自主匯出或控制資料

## 意義
Brain 標誌著 Perplexity Computer 從「無狀態」走向「有狀態」AI Agent，讓 Agent 從工作中學習，而非只記住使用者偏好。
